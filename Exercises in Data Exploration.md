</ul>

  ROUND(STDDEV(measure_value),<span class="pl-c1">2</span>) <span class="pl-k">AS</span> stddev_value
<span class="pl-k">FROM</span> <span class="pl-c1">health</span>.<span class="pl-c1">user_logs</span>
<span class="pl-k">WHERE</span> measure <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">'</span>blood_glucose<span class="pl-pds">'</span></span>)

<span class="pl-k">SELECT</span> 
  ROUND((mean_value <span class="pl-k">-</span> mode_value) <span class="pl-k">/</span> stddev_value,<span class="pl-c1">2</span>) <span class="pl-k">AS</span> pearson_corr_1,
  ROUND(<span class="pl-c1">3</span> <span class="pl-k">*</span> (mean_value <span class="pl-k">-</span> median_value) <span class="pl-k">/</span> stddev_value,<span class="pl-c1">2</span>) <span class="pl-k">AS</span> pearson_corr_2
<span class="pl-k">FROM</span> blood_glucose_cte;</pre></div>

<p>ROW_NUMBER() orders each value depending on its position in the bucket, regardless of whether there are duplicates. Equation = N + 1.</p>
<p>RANK() ensures that duplicates have equal position, where the subsequent position after a set of duplicates is N + X (where X = number of duplicates).</p>
<p>DENSE_RANK() also ensures that duplicates have equal position, but the subsequent position after a set of duplicates is N + 1.</p>
